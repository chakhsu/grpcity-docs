# Get Started

我们假设你已经入门了 gRPC、Node.js 和 JavaScript，有一个基础的认知。官方文档是最好的老师，关于基础 gPRC 入门推荐：[link](https://github.com/grpc/grpc-node/tree/master/examples/helloworld/dynamic_codegen) 。

快速入门教程将会分成 3 篇，最终带你们完成一个完整的基于 gRPCity 的微服务服务端开发与客户端调用。

入门教程对应的项目代码已经放到了GitHub：[link](https://github.com/chakhsu/grpcity-basic-demo) 。
